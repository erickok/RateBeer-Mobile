Release 1.4.0 - New Years Celebration
posted Dec 30, 2009 9:54 PM by Jesper Fussing MÃ¸rk

I just uploaded a brand new version of RateBeer Mobile on the Android Market.

This new release contains two major new features:
- Support for scanning barcodes when doing searches
- Support for a Location Aware list of Beer places. It will display a list of nearby (within 15mi) beer places.

Both these features should be considered beta features so please report any bugs you may encounter.

The barcode scanning feature is very much dependent on the barcodes available in the database and so far only a very small amount of beers are currently in the database. A later version will add support for adding barcodes to the database.

Enjoy and Happy New Year!!

Development News
2009-11-02

Although RateBeer Mobile has not seen any updates in a while its because I've been busy creating a test suite to give the application a thorough testing.

I've had almost no bug reports so its hard to fix "Force Close" issues as people report on the Market. To fix these bugs I will need more valid bug reports including: Device, Android Version, Which screen, What was the user trying to do and so on. As long as I don't get any valid bug reports I won't fix any bugs unless I happen to stumble upon them myself. Hence the test suite. Hopefully the test suite will help me find bugs as well as provide a way to easily test the application when new versions of Android is unleashed.

I've also been working on a small new feature... So stay tuned.
